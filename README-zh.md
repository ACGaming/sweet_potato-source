## 版本
对于本模组不支持1.15.2及更低版本的问题我们十分抱歉，因为本模组中“魔法锅”方块的合成配方中需要下界更新的新东西。我们会在一段时间之后发布补丁版本。<br />
本模组需要的软件配置有：
* Fabric api 0.13.1+build.370；
* Minecraft 1.16.1（我们不确定1.16(.0)版本是否兼容）；
* Fabric loader（最好是0.8.9+build.203）。
