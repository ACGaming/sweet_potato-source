## 版权
teddyxlandlee及烤地瓜项目组拥有该模组的版权。<br />
任何人都可以对此模组提出意见和建议。
## 版本
对于本模组不支持1.15.2及更低版本的问题我们十分抱歉，因为本模组为新版下界风格。<br />
本模组需要的软件配置有：
* Fabric api；
* Minecraft 1.16.1（我们不确定1.16的其他版本是否兼容）；
* Fabric loader（0.9.0+）。
